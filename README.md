🌐 Interactive Sales Deck — The Dubai Mall
🚀 Overview
This project is a cinematic, interactive web-based sales deck designed to showcase The Dubai Mall as a global destination for retail, entertainment, and business opportunities.

It replaces traditional presentations with a high-end, immersive digital experience that allows users to explore independently.

🎯 Objectives
Create instant visual impact

Deliver a premium storytelling experience

Drive:

Retail leasing opportunities
Sponsorship partnerships
Event bookings
✨ Key Features
🎬 Cinematic Hero Section
Fullscreen background video
Smooth animations using Framer Motion
Luxury UI inspired by Apple, Tesla
🧭 Interactive Navigation
Non-linear exploration
User-controlled journey
🏙️ Experience Sections
Why This Property

⚡ Performance Optimized
Lazy loading
Optimized video (WebM + MP4)
Poster-first loading strategy
Minimal blocking resources
🛠️ Tech Stack
Frontend: TypeScript
Styling: Tailwind CSS
Animations: Framer Motion

▶️ Setup Instructions
# Clone the repository
git clone https://github.com/Mylavarapu-Sowmya/interactive-sales-deck-dubai-mall.git

# Navigate into project
cd interactive-sales-deck-dubai-mall

# Install dependencies
npm install

# Run development server
npm run dev
