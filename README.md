# 🌐 Dubai Mall Interactive Sales Deck

## 🚀 Overview
This project is a **cinematic, interactive sales deck** designed to position *The Dubai Mall* as a global destination for retail, entertainment, and business opportunities.

It replaces traditional slide-based presentations with a **digideck-style immersive experience**, enabling users to explore the content independently through a structured, presentation-driven flow.

---

## 🎯 Objectives
- Create **instant visual impact within seconds**
- Deliver a **premium, cinematic storytelling experience**
- Drive business actions:
  - Retail leasing opportunities  
  - Sponsorship partnerships  
  - Event bookings  

---

## ✨ Key Features

### 🎬 Cinematic Hero Experience
- Fullscreen background video with poster-first loading  
- Smooth animations using Framer Motion  
- Luxury UI inspired by Apple & Tesla  

### 🧭 Interactive Deck Navigation
- Non-linear exploration (digideck-style)  
- Floating index + section-based navigation  
- Smooth transitions between sections  

### 🏙️ Structured Deck Sections
- Why This Property  
- Retail & Luxury  
- Dining & Lifestyle  
- Attractions & Entertainment  
- Events & Platform  
- Call-to-Action  

Each section functions like a **presentation slide**, delivering a focused business message.

---

## ⚡ Performance Optimization
- Lazy loading for videos and images  
- WebM + MP4 video support  
- Poster-first rendering for faster LCP  
- Optimized assets for smooth performance  

---

## 🛠️ Tech Stack
- **Frontend:** React (TypeScript)  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Build Tool:** Vite  
- **Deployment:** Vercel / Netlify
